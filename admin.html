Captains' Weekly Checklist

Everything a captain needs to do around a match, in order. League links need the club login for the Cherwell League site.

Before the Game

  1. Pick a team.
  2. Check registrations. Cherwell League — club details Log in and go to Registrations on the top bar. You'll need the club login.
  3. Send the teams out.
  4. Make sure the iPad is charged.

After the Game

  1. Submit the result — home team only. Submit result
  2. Submit the scorecard — home team only. Submit via Play-Cricket score Make sure the scorecard is edited correctly and every player is included. To edit it first: log in here and go to Matches → Edit Scorecard. If the game was NOT scored on Play-Cricket, it needs to be next time. For this week, input the scorecard manually here.
  3. Write the captain's report. Captain's report form
  4. Make sure the umpire reports. This is the captain's responsibility. Send the umpire this link: umpire feedback form.
  5. Text a short match report to Tom Bendall, who will upload it to the website.

Spotted a mistake in a scorecard? Submit a problem here.
